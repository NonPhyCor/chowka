# Repogenesis - FinTech Track

Welcome to the **FinTech Track** of Repogenesis!  
This repository acts as the **foundation** for all projects related to finance, digital payments, analytics, and financial technology.

Participants will **fork this repository**, create a folder with their **team name**, and build their project inside it.

## What Is FinTech?

FinTech focuses on building innovative solutions around:

- Digital payments (like UPI-style flows)
- Wallets & money management
- Personal finance tools
- Budgeting apps
- Expense sharing
- Fraud detection
- Insurance tech
- Investment analysis
- Market prediction tools
- Financial dashboards
- Ledger & accounting systems
- AI-powered finance insights

You may work on **any financial domain** of your choice.

## How to Participate

1. **Fork** this repository.
2. **Clone** your fork.
3. Create a new folder with your **team name** inside the root directory: `/Your-Team-Name/`

4. Build your entire project inside this folder.
5. Push to your forked repo.
6. Submit the link according to event instructions.

## Domain Guidelines

FinTech apps deal with **sensitive data**. Follow these best practices:

- Do **not** use any real personal data.
- Use mock data or the dummy data provided.
- Avoid hardcoding secret keys or credentials.
- Keep sensitive logic server-side where possible.
- Use proper number handling for money (avoid float errors).

## Resources & Starters

The `starter/` and `docs/` folders include:
